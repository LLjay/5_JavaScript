window : 자바 스크립트의 내장 객체, 브라우저 창이 열릴 때마다 하나씩 만들어짐 document : 웹 문서(html)마다 하나씩 만들어지는 객체 자바 스크립트는 .을 이용해 체인 형식으로 기술해 어떤 요소 안의 속성을 불러올 수 있음 divE1.style.backgroundColor = "black"; divE1.innerHTML = "안녕하세요"; divE1.onclick = function(); -> 이벤트 핸들러 input -> 키 값 형태로 저장, name이 key value가 value querySelector : 아이디, 클래스 요소 상관 없이 제대로 써주기만 하면 모든 요소를 가져올 수 있음 / 자식 속성도 마찬가지 const : 상수를 표현하기 위한 자료형, 한 번 지정하면 값을 변경할 수 없음 (속성은 변경 가능) Naming Type 1. camelCase 2. snake_case 3. kebab-case
